Project Closure

ISO Clause: 8.5 PR15 Version : 3.00 Date : 17th May 2010

15.1 Introduction

15.1.1 Purpose

The purpose of this procedure is to describe the activities to be performed at the time of project closure.

15.1.2 Policy

All projects will prepare the project closer report.

15.1.3 Scope

This procedure will be applicable during the closure of any projects.

15.1.4 Entry Criteria

This process will start after the project delivery and acceptance given by the customer.

15.2 Stakeholders

15.2.1 Responsibilities

Sl. No. Role Responsibilities Authority
1. PM Collect Project Metrics, Prepare Project Closure Report, Conduct Project Closure Meeting, Archive Documents and Source Code NIL
2. QM Review the project closure report and the project metric Approve the closure report

15.3 Task Details

15.3.1 Collect Project Metrics

The PM will collect the Project/sub-project metrics as per Project Metrics Report. The metrics collected will be analyzed by SQA using appropriate statistical techniques.

15.3.2 Prepare Project Closure Report

The PM will prepare Part - A of Project Closure Report (FRPRC01). The PM will invite the following members for the Project Closure meeting:

· Delivery Head

· Sales partner of the project

· Manager Quality

· EPG Members

· System Admin

· Head Training

· Head Finance (Optional)

· Personnel In-Charge (Optional)

· Training In-Charge (Optional)

· Head Documentation (Optional)

15.3.3 Conduct Project Closure Meeting

The minutes of meeting will be recorded in PCC. If PCC is not available then the Minutes of meeting will be recorded using FRMMR02

The objective of the Project Closure meeting is:

· To analyze the project execution

· To discuss Project Learning

· To discuss any specific improvement from the support groups

The PM will hand-over the project folder to SQA for maintaining in the Organization Process database.

15.3.4 Archive Documents and Source Code

The PM will archive documents and source code as per the Backup, Restore and Archival Maintenance (PR10) procedure
