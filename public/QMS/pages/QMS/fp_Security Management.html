Security Management
ISO Clause: 6.4 FP07 Version: 3.00 Date: 17th May 20107

7.0 Introdution

7.1.1 Purpose
The purpose of this procedure is to define the policies for Security Management.

7.1.2 Policy
This process will be applicable for maintaining the security of the premises and the facility of the organization.

7.1.3 Scope
This procedure addresses the security of the Office premises and the data / information handled in Integra.

7.3 Process Description

7.3.1 Internet Security
The Head- Engineering will ensure the following:

Inbound PolicyInbound IP services such as Telnet and FTP are disabled.

Outbound PolicyOutbound Telnet services are disabled.

For any Inbound or Outbound services that are disabled, the requestor will raise a request through PCC or Job Request (FRIFR03) and submit it to PM / DH for approval. The Infrastructure team will service the approved request.

7.3.2 Router Security

The System Administrator will ensure that all the routers are secured with user passwords and "enable" passwords. The System Administrator will manage these passwords and a list of the same is handed over to the Head- Engineering. The router configuration details will be updated once in 3 months, if required or whenever routing configuration changes.

7.3.3 Virus Protection


The System Administrator will ensure that all the workstations will have approved anti-virus software installed.

Automatic update will be enables on all systems. In the event that automatic update is not available for the installed anti-virus software, the System Administrator will provide updates/patches for the anti-virus software, which will be intimated and made available to the employees. The employees are responsible for updating their workstations with the new anti-virus updates/patches.

The System Administrator will verify the effectiveness of the anti-virus software updates/patches during the Software Audit.

7.3.4 Un-authorized Software Policy

The employees are not allowed to download or install any unauthorized software from the Internet or from any other source.

The System Administrator is responsible for installing all the software in the organization.

The System Administrator will audit workstations once in three months to ensure that no unauthorized software is being used. The result of the audit will be recorded using Software Audit Report (FRSEC01).

If any usage of unauthorized software is found, the same would be deleted and the Functional Head would take appropriate action against the employee.

However, employees can use any freeware available on the corporate Intranet.

7.3.5 Physical Security

The General Admin In-Charge will ensure that the Development Center will have round-the-clock manned security personnel.

The System Administrator will ensure that the Server Rooms have restricted access.

7.3.6 Safety Procedure

Safety precautions have been in place in the office premise. Fire alarm systems have been installed in the office premises and fire extinguishers are placed in select locations of the building. The General Admin personnel are trained in safety measures to follow in case of any accident. The General Admin In-Charge will verify proper functioning of all the safety instruments installed in the premises in January every year.

The following steps are to be followed in case of fire alarm:

  • All the Admin will reach the Admin Section on hearing the fire alarm siren.
  • Security personnel will identify the zone and raise an alarm informing the Admin In-Charge and any admin personnel in the admin area.
  • Admin In-Charge will enable the emergency exit in the access system to open all doors
  • Technical In-Charge will reach the switchgear room and immediately switch off the power supply pertaining to the identified zone.
  • The housekeeping boys along with the housekeeping Supervisor will ensure that all the doors are open and fire marshal in each section should alert the employees and force them to move outside the building.
  • Security personnel after identifying the zone will alert other guards and open the back exit leaving one guard at the post.
  • Housekeeping boys will reach the fire zone with the extinguishers / sand buckets and douse the fire.
  • The Admin In-Charge will ensure that the fire extinguisher are refilled on a regular basis and firs safety steps will be displayed at prominent location of the building.
  • The Admin In-Charge will ensure that the employees are trained on safety precautions.
